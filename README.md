A full-stack **MERN** (MongoDB, Express, React, Node.js) e-commerce application with user authentication, product management, cart & checkout flow, and a basic admin panel. This README contains setup, run and deployment instructions, environment variable examples, and common commands.

---

## 🔎 Features
- User authentication (JWT + refresh tokens)  
- Product listing, search, filtering, and categories  
- Product detail pages with reviews  
- Shopping cart & checkout flow (Stripe / mock)  
- Order history & order management  
- Admin panel: CRUD for products & orders  
- RESTful API (Express) + React frontend  
- Client-side form validation and responsive UI
